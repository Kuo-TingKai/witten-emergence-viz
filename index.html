01

資訊局域觀測限制 · Type III₁ 代數

傳統量子力學假設希爾伯特空間可張量分解，於是有部分跡與密度矩陣：

$$\mathcal{H}=\mathcal{H}_A\otimes\mathcal{H}_B,\qquad S=-\mathrm{Tr}(\rho_A\ln\rho_A)$$

但在相對論性 QFT 中，Reeh–Schlieder 定理告訴我們：真空在任何空間邊界上都帶有無限的紫外糾纏。局域觀測者觸及的，是一種沒有跡、沒有局域密度矩陣的 Type III₁ von Neumann 代數。

Type III 的致命特性

  • 沒有跡 (Trace)
  • 沒有局域密度矩陣 $\rho_A$
  • 單一時空區域的糾纏熵在數學上未定義（等同無限大）

黑洞外、或 de Sitter static patch 中的觀測者，因視界遮蔽，只能存取 Type III 代數的可觀測量。

02

代數過渡 · 微擾重力打開「跡」的開關

當牛頓常數 $G_N>0$（全像中的 $1/N$ 微擾）被引入，重力對觀測者的哈密頓量進行幾何著裝 (Gravitational Dressing)。把重力約束（微分同胚不變性）強加到代數上，數學上等同一次模交叉積 (Crossed Product) —— Type III 因此「降級」為 Type II，跡與密度矩陣重新出現。

跡重現 → 廣義熵的具現化

有了跡，就能定義正規化的代數糾纏熵。Witten 證明它正是黑洞的廣義熵：

$$S_{\text{gen}}=\frac{\text{Area}}{4G_N}+S_{\text{out}}$$

宏觀的黑洞熱力學、霍金輻射、熱效應，本質上是觀測者引入重力後，原本被 Type III 遮蔽的龐大資訊，在過渡到 Type II 時所付出的代數代價。

03

四種湧現 + 質量 · 宏觀物理 = 代數資訊流

時間

模自同構流 · Modular Flow

Type III 代數本身沒有客觀時間。但給定局域狀態 $\omega$，Tomita–Takesaki 理論必然唯一確定一參數自同構群：

$$\sigma_t^{\omega}(A)=\Delta^{it}\,A\,\Delta^{-it}$$

參數 $t$ 在宏觀上正是觀測者的固有時間。時間的流逝，是代數為維持統計平衡而自激的演化流（Connes–Rovelli 熱時假說）。

空間

子代數包含 · 糾纏網絡

空間區域 $U$ ↔ 邊界算符子代數 $\mathcal{A}(U)$。遠近、大小、包含關係 ↔ 子代數的包含與對易：

$$A\subset B \;\Longleftrightarrow\; \mathcal{A}(A)\subset\mathcal{A}(B)$$

糾纏楔重構：糾纏越強，宏觀距離越近；熵歸零，空間結構就斷裂。

熱（熵）

Type III → Type II 的跡效應

引入微擾重力後代數過渡所重現的跡，使原本被視界遮蔽的資訊以熵的形式具現：

$$S_{\text{gen}}=\frac{\text{Area}}{4G_N}+S_{\text{out}}$$

霍金溫度、黑洞熱力學，皆為此代數代價的宏觀表象。

能量

模哈密頓量擾動 · 幾何阻力

模哈密頓量 $K$ 是代數內在時間流的生成元（Rindler 中即 Boost 能量）。擾動糾纏態：

$$\Delta S=\Delta\langle K\rangle$$

在 AdS/CFT 中對應 Ryu–Takayanagi 極小曲面的面積變化 —— 宏觀幾何對抗糾纏擾動的「彈性」，就是能量。

質量

能量聚集 · Nambu–Goto 弦振動的表象

質量是局域能量的凝聚（$E=mc^2$）。當能量在宏觀時空高度聚集形成時空缺陷，弦/膜的運動由 Nambu–Goto 作用量主導。在代數底層，這代表大量 Type II 自由度被「緊緻化／凝聚」在特定時空邊界（弦張力、原子質量），成為觀測者測得的靜止質量。

幾何 · 拓撲

面積 = 糾纏

Ryu–Takayanagi · 幾何橋樑

時空的連續性，究竟是哪來的？

$$S(A)=\frac{\text{Area}(\gamma_A)}{4G_N}$$

邊界區域 $A$ 的馮紐曼糾纏熵，等於內部極小曲面 $\gamma_A$ 的幾何面積。糾纏歸零 → 面積歸零 → 時空直接斷開。連續性是被糾纏硬生生「拉」出來的。

重構 = 糾錯

JLMS / ADH · 子區域對偶

補集全毀，內部還能完整還原？

$$\hat H_{\text{mod}}^{\partial}=\frac{\hat{\text{Area}}(\gamma_A)}{4G_N}+\hat H_{\text{mod}}^{\text{bulk}}$$

落在糾纏楔內的 bulk 時空點，被「加密」在邊界區域 $A$。即使補集遭嚴重噪聲（抹除錯誤）破壞，仍能單憑 $A$ 的算子完整還原。Bulk = 抗抹除的 code subspace。

拓撲 = 邏輯算子

TQFT · 低能極限 · Chern–Simons

算子怎麼拉扯、怎麼形變，測量值都不變 —— 為什麼？

$$\dim V(\Sigma)=Z(\Sigma\times S^1)$$

把 bulk 重力推進到低能極限，度規動力學被凍結，留下純拓撲項（Chern–Simons）。其上的 Wilson loop 是拓撲算子，形變不改測量值 —— 這恰恰是 QEC 邏輯算子的特徵：對局部噪聲完全免疫。TQFT 說「拓撲所以不變」，QEC 說「邏輯所以不錯」，兩句話透過函子 $V:\text{Bord}\to\text{Vec}$ 在數學上融合。
